Enhancement One: Software Design and Engineering

Artifact: CS-499, 3-2 Milestone Document
Description: This narrative explains enhancements made in software design and engineering.

This enhancement focuses on improving the structure and functionality of a medical scheduling application. Originally developed in CS-320, the application allowed users to schedule appointments and store contact information. The enhancement involved extending JUnit tests for broader test coverage, adding new data validation features, and refining search functionality for better usability. These changes showcase my ability to design efficient, testable, and user-friendly software solutions.

Enhancement Three: Databases

Artifact: Database enhancements on Weight Tracker
Description: Added database capabilities and integration for dynamic data handling.

This enhancement involved adding SQLite database integration to a 3D rendering project initially created in CS-330. The database replaced hardcoded transformation values, allowing for dynamic runtime updates to objects within a desk scene. This included creating database schemas, performing indexed queries, and efficiently retrieving data. The project highlights my ability to integrate databases into computational graphics to enable dynamic and scalable solutions.
